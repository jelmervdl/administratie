Header

Header()

Versie

1.0

Omschrijving

Deze methode wordt gebruikt om de koptekst in te stellen. Het wordt automatisch aangeroepen door AddPage() en hoeft niet direct door de toepassing aangeroepen te worden. De toepassing in FPDF is leeg, het is dus nodig om deze via een subclass aan te roepen en de methode over te slaan als je een specifieke weergave wilt.

Voorbeeld

class PDF extends FPDF
{
function Header()
{
    //Selecteer Arial vet 15
    $this->SetFont('Arial','B',15);
    //Beweeg naar rechts
    $this->Cell(80);
    //Titel in cel met rand
    $this->Cell(30,10,'Title',1,0,'C');
    //Line break
    $this->Ln(20);
}
}
