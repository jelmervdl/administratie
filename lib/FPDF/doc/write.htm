Write

Write(float h, string txt [, mixed link])

Versie

1.5

Omschrijving

Deze methode geeft een tekst weer vanaf de huidige positie. Wanneer de rechtermarge is bereikt (of het \n karakter gevonden is) wordt een line break uitgevoerd en gaat de tekst verder vanaf de linker marge. Wanneer het commando stopt, wordt de huidige positie aan het einde van de tekst gelaten.
Het is mogelijk om een link op de tekst te plaatsen.

Parameters

h
Regel hoogte.
txt
String om weer te geven.
link
URL of identifier die door AddLink() wordt teruggegeven.

Voorbeeld

//Begin met een normaal lettertype
$pdf->SetFont('Arial','',14);
$pdf->Write(5,'Visit ');
//Plaats dan een blauw onderstreepte link
$pdf->SetTextColor(0,0,255);
$pdf->SetFont('','U');
$pdf->Write(5,'www.fpdf.org','http://www.fpdf.org');
