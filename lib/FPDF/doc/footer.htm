Footer

Footer()

Versie

1.0

Omschrijving

Deze methode wordt gebruikt om de pagina voettekst te maken. Het wordt automatisch aangeroepen door AddPage() en Close() en hoeft niet direct aangeroepen te worden door de toepassing. De toepassing in FPDF is leeg, het is dus nodig om deze via een subclass aan te roepen en de methode over te slaan als je een specifieke weergave wilt.

Voorbeeld

class PDF extends FPDF
{
function Footer()
{
    //Ga naar 1,5 cm van de onderkant
    $this->SetY(-15);
    //Selecteer Arial cursief 8
    $this->SetFont('Arial','I',8);
    //Geef gecentreerd weer het pagina nummer
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
