Output

string Output([string name [, string dest]])

Versie

1.0

Omschrijving

Stuurt het document naar de gegeven bestemming: String, lokaal bestand of browser. In het laatste geval wordt of de plug-in gebruikt (wanneer aanwezig) of een download ("Opslaan als" dialoog scherm) geforceerd.
Deze methode roept eerst Close() aan wanneer nodig om het document te beëindigen.

Parameters

name
De naam van het bestand. Wanneer niet gegeven zal het document naar de browser worden verstuurd. (Bestemming I) met de naam doc.pdf.
dest
Bestemming waar het bestand naar toe moet worden verstuurd. Het kan een van de volgende waarden aannemen:
  • I: Verstuurt het bestand naar de browser. De plug-in wordt gebruikt wanneer beschikbaar. De naam ingegeven door name wordt gebruikt wanneer men de "Opslaan als" optie gebruikt bij de link die de PDF genereert.
  • D: Verstuurt het naar de browser en forceert een download met de naam gegeven door name.
  • F: Slaat het op als een lokaal bestand met de naam gegeven door name.
  • S: Geeft het document terug als een string. name wordt genegeerd.
Als de parameter niet gespecificeerd is maar een naam is wel gegeven, dan is de bestemming F. Als er helemaal geen parameter gespecificeerd is, is de bestemming I.

Opmerking: Om goed te kunnen werken met vorige versies wordt ook een boolean waarde geaccepteerd. (false voor F en true voor D).
