AddFont

AddFont(string family [, string style [, string file]])

Versie

1.5

Omschrijving

Importeert een TrueType of Type1 lettertype en maakt het beschikbaar. Het is noodzakelijk eerst een definitie van het lettertype te genereren met het makefont.php programma.
Het definitie bestand (en het lettertype bestand zelf wanneer deze wordt ingevoegd) moet aanwezig zijn in de font directory. Als het niet gevonden kan worden, zal de fout "Could not include font definition file" (= "Kan het lettertype definitie bestand niet toevoegen") worden gegenereerd.

Parameters

family
Lettertype Familie. De naam kan willekeurig gekozen worden. Als het een standaard familie naam is, zal het het huidige lettertype overslaan.
style
Stijl van het lettertype. Mogelijk waardes zijn (hoofdletter ongevoelig):
  • lege string: normaal
  • B: Vet
  • I: Cursief
  • BI of IB: Vet Cursief
De standaard waarde is normaal.
file
Het lettertype definitie bestand.
Standaard, de naam is opgebouwd uit de familie en stijl, in kleine letters zonder spaties.

Voorbeeld

$pdf->AddFont('Comic','I');
is gelijk aan:
$pdf->AddFont('Comic','I','comici.php');
