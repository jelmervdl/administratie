Image

Image(string file, float x, float y [, float w [, float h [, string type [, mixed link]]]])

Versie

1.1

Omschrijving

Geeft een afbeelding weer. De linkerbovenhoek met worden gegeven. De dimensies kunnen worden gespecificeerd op verschillende manieren:
  • Expliciet breedte en hoogte invoeren (weergegeven in de eenheid van de gebruiker)
  • Een expliciete dimensie, de ander wordt automatisch berekend om de verhoudingen te behouden
  • Geen expliciete dimensies. In dit geval wordt de afbeelding weergegeven op 72 dpi
Ondersteunde bestandstypen zijn JPEG en PNG.

Voor JPEG zijn alle kleuren toegestaan:
  • Grijswaarden
  • Ware Kleuren (24 bits)
  • CMYK (32 bits)
Voor PNG zijn de volgende types toegestaan:
  • Grijswaarden van maximaal 8 bits (256 levels)
  • Geïndexeerde kleuren
  • Ware kleuren (24 bits)
Maar niet:
  • Interlacing
  • Alpha kanaal
If a transparent color is defined, it will be taken into account (but will be only interpreted by Acrobat 4 and above).
The format can be specified explicitly or inferred from the file extension.
It is possible to put a link on the image.

Remark: if an image is used several times, only one copy will be embedded in the file.

Parameters

file
Naam van het bestand waarin de afbeelding zich bevindt.
x
Abscis (X-Coördinaat) van de linkerbovenhoek.
y
Ordinaat (Y-Coördinaat) van de linkerbovenhoek.
w
Breedte vam de afbeelding op de pagina. Wanneer niet ingegeven of gelijk aan nul, wordt deze automatisch berekend.
h
Hoogte vam de afbeelding op de pagina. Wanneer niet ingegeven of gelijk aan nul, wordt deze automatisch berekend.
type
Bestandstype. Mogelijke waardes zijn (hoofdletter ongevoelig): JPG, JPEG, PNG. Wanneer niet ingegeven, wordt het type bepaald uit de bestandsextensie.
link
URL of identifier die door AddLink() wordt teruggegeven.
