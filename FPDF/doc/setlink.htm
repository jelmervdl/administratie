SetLink

SetLink(int link [, float y [, int page]])

Versie

1.5

Omschrijving

Stelt de pagina en de positie in waar de link naartoe verwijst.

Parameters

link
URL of identifier die door AddLink() wordt teruggegeven.
y
Ordinaat (=Y coördinaat) van de positie van de bestemming; -1 geeft de huidige positie aan. De standaard waarde is 0 (bovenkant van de pagina).
page
Nummer van de pagina van de bestemming; -1 geeft de huidige pagina aan. Dit is ook de standaard waarde.
